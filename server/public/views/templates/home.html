How To Use This Application

This application is meant for those in need and are looking to utilize local resources.

The resources include healthcare:

  • Women's Health

  • Mental Healthcare

As well as:

  • Other Services

To search and find resources please either select one of the resource groups

or select from the above navigation bar

About The Creator And Mission Statement

This application was inspired by Krista J Van Eps and developed by Laura A Goetz.

Krista is a hard-working student at Metro State University and is currently studying Social Work.

She expressed concern because of the lack of available way to find resources for people that are in need of them.

Which is where I come in, I took Krista's concerns and created a tool for users

to be able to find local resources available to them.

The mission for this application is to provide users with an easy and unbiased way

to locate, contact and learn more about their options.
