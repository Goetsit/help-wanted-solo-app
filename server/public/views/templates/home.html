How To Use This Application

Welcome and thank you for using this resource. Help Wanted is here to help you find the right resources for you and your loved ones. Please create an account or log in to begin utilizing the database of resources. Once you have an account you can navigate to the Resources tab. This tab will provide users with a brief description of resources locally and nationwide.

These resources include healthcare options such as:
  • Women's Health

  • Mental Health

  • Family and Individual Services

Your user tab will list bookmarked resources as well as providing users the option to add new Resources and a some details about their user as well.

Please keep in mind, Help Wanted does not endore any of these resources, this is an open source web application and usage of these resources is up to the discretion of the individual in need.

About The Creator And Mission Statement

This application was inspired by Krista J Van Eps and developed by Laura A Goetz.
Krista is a hard-working student at Metro State University and is currently studying Social Work.
She expressed concern because of the lack of available way to find resources for people that are in need
of them. Which is where I come in, I took Krista's concerns and created a tool for users
to be able to find local resources available to them.
The mission for this application is to provide users with an easy and unbiased way
to locate, contact and learn more about their options.
